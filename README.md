# ♓ CONSULTOR DE SIGNO ♒

Um site desenvolvido para obtenção de nota no portifolio do 5 semestre da UNOPAR usando tecnologias PHP e bootstrap com estilização em css.

## ✨ FUNCIONALIDADES ✨

- O usuário seleciona a data desejada;
- Ao clicar em Enviar, o site mostrará o signo correspondente à data selecionada;
- Caso o usuário deseje consultar outra data, ele deverá selecionar o botão Voltar, onde será redirecionado novamente para a página em que poderá escolher outra data.
